# 🎮 Simon Says Game

A classic **Simon Says** memory game built with vanilla HTML, CSS, and JavaScript.

## How to Play

1. **Press any key** to start the game.
2. Watch the button that flashes — remember the sequence!
3. Click the buttons in the **same order** as the sequence.
4. Each round adds one more color to the sequence.
5. If you click the wrong button, the game ends and shows your score.
6. Press any key to **restart** and try to beat your high score!

## Tech Stack

- **HTML** — Game layout and structure
- **CSS** — Responsive styling with media queries
- **JavaScript** — Game logic, sequence generation, and input handling
